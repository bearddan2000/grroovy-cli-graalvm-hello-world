# grroovy-cli-graalvm-hello-world

## Description
A POC for graalvm using native-image to build a standalone executable binary.

## Tech stack
- groovy
- maven
- native-image

## Docker stack
- gradle
- ghcr.io/graalvm/graalvm-ce:ol8-java11-22.3.3

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
